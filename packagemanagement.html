What is a package manager?

A package manager is a program that... well... manages packages. A package is a bundle of software that a package manager can find and install. Think of a package manager like the App Store, with packages being the apps.

Why are package managers useful?

Package managers allow you to easily find and install any software you want. If you've use Windows, you would know how annoying it is to search for an installer for a program, download the installer, run the installer, and then have to delete the installer after to free up space. Package managers streamline this process by doing it all for you.

Package managers also have functionality to update your entire system with a single command, meaning you no longer need to go through every app that needs an update to perform maintanence.

Common Package Managers

Install Command:

Package Manager Command Example (installing firefox) Distro(s) Package Search Website(s)
apt apt install firefox Debian, Ubuntu, Mint Debian Package Search, Ubuntu Package Search
dnf dnf install firefox Fedora, Red Hat Fedora Package Search
pacman pacman -S firefox Arch, CachyOS Arch Package Search
yay yay -S firefox Arch User Repository AUR Package Search
flatpak flatpak install firefox Distro independant Flathub
portage emerge --ask www-client/firefox-bin Gentoo Gentoo Package Search
NixOS nix-env --install firefox #This is a temporary install on nix, use the configuration file for a permenant install NixOS Nixpkg Search
Snap snap install firefox Ubuntu Snapcraft

Update Command:

Package Manager Command Example Distro(s)
apt apt update && apt upgrade Debian, Ubuntu, Mint
dnf dnf upgrade Fedora, Red Hat
pacman pacman -Syu Arch, CachyOS
yay yay -Syu Arch User Repository
flatpak flatpak update Distro independant
portage emerge --sync && emerge --update --deep --with-bdeps=y @world Gentoo
NixOS nixos-rebuild switch --flake ~/flake#hostname NixOS
Snap snap refresh Ubuntu
